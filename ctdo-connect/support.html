To create an API key for use with CTDO Connect:

  1. Sign in to App Store Connect
  2. Go to Users and Access
  3. Select the Keys tab under "Integrations"
  4. Click the + button to create a new key
  5. Give it a name (e.g., "CTDO Connect")
  6. Select Admin or Developer access (Admin recommended)
  7. Download the .p8 private key file immediately (you can only download it once!)
  8. Note the Key ID and Issuer ID from the page

You'll need all three pieces of information (Issuer ID, Key ID, and the .p8 file) to add the key to CTDO Connect.

Yes, your API key is stored securely. Here's how we protect it:

  • iOS Keychain: Your private key is stored in the iOS Keychain, which provides hardware-level encryption
  • No Cloud Sync: We don't sync your keys to iCloud or any cloud service
  • No External Servers: We don't have any servers - all communication is directly between your device and Apple
  • Sandboxed Storage: The app's data is sandboxed and cannot be accessed by other apps

CTDO Connect offers two ways to create certificates:

Option 1: Auto-Generate (Recommended)

  1. Go to Certificates tab and tap +
  2. Select certificate type
  3. Enter your email address
  4. Tap "Auto-Generate CSR + Private Key"
  5. The app will generate both automatically

Option 2: Import existing CSR

  1. On your Mac, open Keychain Access
  2. Go to Keychain Access ‚Üí Certificate Assistant ‚Üí Request a Certificate from a Certificate Authority
  3. Enter your email, select "Saved to disk"
  4. Save the .certSigningRequest file
  5. Import this file in CTDO Connect

If you're not seeing your resources, try these steps:

  1. Check API Key Permissions: Ensure your API key has "Admin" or "Developer" access level
  2. Pull to Refresh: Pull down on any list to force a refresh from Apple's servers
  3. Check Team Selection: If you have multiple teams, make sure you're using the correct API key
  4. Wait for Propagation: Sometimes Apple's API takes a few minutes to reflect recent changes
  5. Clear Cache: Go to Settings ‚Üí Clear Cache, then refresh

Yes! CTDO Connect supports editing profiles. Here's how it works:

Apple's API doesn't allow direct editing of profiles. So when you "edit" a profile in CTDO Connect, the app:

  1. Saves your new settings
  2. Deletes the old profile
  3. Creates a new profile with your updated settings

This happens seamlessly - you'll see a loading indicator while the process completes. The new profile will have a new UUID, which is normal.

CTDO Connect supports multiple API keys for different teams:

  1. Go to Settings tab
  2. Tap "Add API Key"
  3. Enter the credentials for your other team
  4. Once added, tap on any key to select it as active

The app will immediately load data for the selected team. A checkmark indicates which key is currently active.

The Background Updates feature allows CTDO Connect to:

  • Periodically check your device registration status
  • Notify you when a device status changes (e.g., Processing ‚Üí Enabled)

This is especially useful when you register a new device and want to know when it's ready to use in provisioning profiles.

Note: This feature requires notification permissions to work.
